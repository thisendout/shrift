What is shrift?

shrift lets you specify tests to run against your environment in a language you already know: bash. It designed to run minimally, without any complex setup, and from anywhere you want to run it using the same test commands you would run from a shell prompt.

Test in 60 seconds

Installation is quick and easy, just curl or wget the script and you're off to the races faster than Nicolas Cage can fall in love with a car. Oh Eleanor.

Minimal dependencies

shrift is designed to run without any complex setup, using only tools widely available. Tests, however, can make use of any tools/scripts in your aresenal.

CLI commands to test suite

Steps to test or troubleshoot a problem from the CLI can be instantly transformed into a spec by placing those commands into a file. No translation necessary.
