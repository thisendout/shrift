Quickstart

Below are instructions on how to install shrift and write and run your first spec test.

# download the script
curl -L -o shrift https://github.com/thisendout/shrift/raw/master/shrift
# make executable
chmod +x shrift

Optionally create and run a new a spec test

# write a passing test
echo "test -x ./shrift" > test_spec.sh
# execute with shrift
./shrift test_spec.sh
